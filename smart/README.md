# Ростелеком — Умный дом, который помогает расти

> Сервис от Ростелеком для управления умным домом, объединяющий устройства в едином интерфейсе и формирующий ответственность у подростков через геймификацию

## О проекте

«Ростелеком Умный дом» решает проблему разрозненных приложений для умных устройств, создавая единый интерфейс, который учитывает интересы подростка и формирует ответственность через геймификацию.

### Ключевая идея

Превращаем заботу о доме в увлекательную игру с уровнями, достижениями и реальными привилегиями. Подросток не «выполняет обязанности», а «прокачивает свой дом».

## Запуск

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload --port 8000
```

Или используйте `start-backend.bat`

Откройте http://localhost:8000 в браузере.

## Возможности

### Управление домом
- 💡 Освещение (яркость, цвет, сцены)
- 🔌 Бытовая техника (статус, энергия)
- 🌡️ Датчики (температура, движение)
- 📹 Камеры и безопасность

### Для подростка
- 🎮 Геймификация: уровни, XP, достижения
- 🎯 Фокус-режим: таймер Pomodoro
- 🌙 Вечерний ритуал: пошаговая подготовка ко сну
- 📋 Ответственность: задачи с очками
- 🏠 Статус дома: контроль приборов

### Новые функции
- 🏆 Система достижений (5 типов)
- 📋 Семейные соглашения с согласованием
- 🔔 Телеком-интеграция (SMS, Push, USSD)
- ⚡ Трекинг экономии энергии

## API Endpoints

### Устройства
- `GET /api/devices` — список устройств
- `POST /api/devices/{id}/toggle` — вкл/выкл
- `GET /api/lights` — светильники
- `GET /api/lights/scenes` — сцены освещения
- `POST /api/lights/scene/{name}` — применить сцену
- `GET /api/appliances` — бытовая техника

### Подростковый дашборд
- `GET /api/teen/dashboard/{member_id}` — дашборд
- `GET /api/teen/tasks` — задачи
- `POST /api/teen/tasks/{id}/complete/{member_id}` — выполнить
- `POST /api/teen/focus/start` — начать фокус
- `GET /api/teen/evening-routine/{member_id}` — ритуалы
- `GET /api/teen/home-status` — статус дома

### Достижения
- `GET /api/achievements` — список достижений
- `GET /api/achievements/member/{member_id}` — достижения пользователя
- `POST /api/achievements/check/{member_id}` — проверить прогресс

### Семейные соглашения
- `GET /api/agreements` — список соглашений
- `POST /api/agreements/{id}/agree/{member_id}` — согласиться

### Телеком-интеграция
- `POST /api/telecom/notifications` — создать уведомление
- `POST /api/telecom/notifications/{id}/send` — отправить
- `GET /api/telecom/notifications` — список уведомлений

## Пользовательские сценарии

### Утро Миши
1. 7:00 — Будильник, свет плавно включается
2. 7:15 — Проверка задач и статуса дома
3. 7:20 — Автоматическая проверка приборов перед уходом

### Уроки Миши
1. 15:00 — Запуск «Фокус-режима» на 45 минут
2. Свет настраивается для концентрации
3. После завершения — +50 XP

### Вечерний ритуал
1. 21:00 — Напоминание о ритуале
2. Пошагово: уроки → свет → техника → двери → отдых
3. Интернет замедляется автоматически
