# Ростелеком — Умный дом, который помогает расти

> Сервис от Ростелеком для управления умным домом, объединяющий устройства в едином интерфейсе и формирующий ответственность у подростков через геймификацию

---

## 1. Название и позиционирование

**«Ростелеком»** — умный дом, который не ограничивает, а помогает становиться самостоятельнее.

Слоган: *«Твой дом — твои правила. Твоя ответственность — твой уровень.»*

---

## 2. Целевая аудитория

- **Основная**: Семьи с детьми 12–16 лет
- **Пользователь-подросток**: Активно пользуется смартфоном, интернетом, умными устройствами
- **Пользователь-родитель**: Хочет научить ребёнка ответственности без жёсткого контроля

### Портрет пользователя

**Михаил, 14 лет, 8-й класс**
- Смартфон + умные устройства + домашний Wi-Fi
- Каждое устройство — отдельное приложение → неудобно
- Хочет: управлять своими цифровыми привычками самостоятельно, чувствовать ответственность за дом, иметь единое «нескучное» приложение

---

## 3. Решаемая проблема

### Проблематика
- «Умные» устройства есть, но все на разных приложениях
- Нет единого интерфейса, учитывающего интересы подростка
- Родительский контроль = ограничение → конфликты
- Ребёнок не чувствует ответственности за порядок в доме

### Решение
**Геймификация ответственности** — превращаем заботу о доме в увлекательную игру с уровнями, достижениями и реальными привилегиями.

---

## 4. Ключевые функции

### 4.1 🎮 Система уровней и прогресса
- **6 уровней**: Новичок → Помощник → Организатор → Мастер порядка → Хранитель дома → Легенда
- **Очки опыта (XP)**: За выполнение задач, экономию энергии, своевременное выключение приборов
- **Достижения**: «Мастер порядка» (50 задач), «Эко-воин» (500 XP), «Ночной страж» (10 ритуалов), «Фокус-мастер» (20 сессий), «Марафонец» (1000 мин фокуса)
- **Реальные награды**: Доп. часы интернета, выбор фильма, бюджет на игры

### 4.2 🏠 «Мой дом» — единый пульт
- Все устройства в одном интерфейсе: свет, климат, безопасность, техника
- Быстрые действия: вкл/выкл света, проверка дверей, запуск фокуса
- Визуальная карта дома с интерактивными элементами
- Статус дома: что включено, что забыли выключить

### 4.3 🎯 «Фокус-режим» — самоорганизация
- Таймер Pomodoro (15/25/45/60 мин) с выбором предмета
- Автоматическая настройка света (яркий, холодный для концентрации)
- Приглушение уведомлений через API оператора
- Статистика: сессии, минуты, серия дней, заработанные XP

### 4.4 🌙 «Вечерний ритуал» — подготовка ко сну
- Пошаговый сценарий: проверка уроков → приглушение света → выключение техники → проверка дверей → отдых
- Автоматизация устройств на каждом шаге
- Достижение «Ночной страж» за своевременный отход ко сну
- Замедление интернета в ночное время через оператора

### 4.5 📋 «Домашний детектив» — контроль забытых приборов
- Умные напоминания о включённых приборах
- Удалённое выключение через приложение
- SMS-уведомления через оператора, если интернет недоступен
- «Эко-рейтинг» дома с бонусами за экономию

### 4.6 🤝 «Семейные соглашения» — без конфликтов
- Правила использования интернета, согласованные всей семьёй
- Прозрачная система ограничений и поощрений
- Механизм согласования: подросток может принять или обсудить
- Достижение «Миротворец» за конструктивное общение

## 5. Интеграция с телекоммуникационным оператором

### 5.1 SMS и Push-уведомления
- Уведомления о забытых приборах (утюг, свет, телевизор)
- Напоминания о задачах и вечернем ритуале
- Экстренные оповещения о безопасности

### 5.2 Управление интернетом
- Автоматическое замедление в ночное время (22:00–7:00)
- Режим «Фокус» с блокировкой развлекательных сайтов
- Родительский контроль через API оператора

### 5.3 USSD-управление
- `*123*1#` — статус дома (что включено)
- `*123*2#` — экстренное выключение всех приборов
- `*123*3#` — получение уведомлений при отсутствии интернета

### 5.4 IoT-платформа оператора
- Подключение устройств через сеть оператора
- Облачное хранение данных и резервное копирование
- Приоритетная поддержка для премиум-пользователей

## 6. Баланс свободы и безопасности

| Аспект | Подросток видит | Родитель видит |
|--------|-----------------|----------------|
| Контроль | Прозрачные правила с возможностью договариваться | Мягкие уведомления, не жёсткие блокировки |
| Награды | Реальные привилегии за ответственность | Автоматическая система поощрений |
| Безопасность | «Дом под контролем» — статус устройств | Экстренные оповещения о проблемах |
| Интернет | «Фокус-режим» = мой выбор, не наказание | Время фокусировки и статистика |

## 8. Пользовательские сценарии

### Сценарий 1: «Утро Миши»
1. **7:00** — Будильник через умную колонку
2. **7:05** — Свет в комнате плавно включается (сцена «Утро»)
3. **7:10** — Кофеварка начинает готовить кофе для родителей
4. **7:15** — Миша открывает «ДомоСферу»: задачи на день, прогноз погоды, статус дома
5. **7:20** — Автоматическая проверка: все ли приборы выключены перед уходом
6. **+10 XP** за проверку дома

### Сценарий 2: «Уроки Миши»
1. **15:00** — Миша запускает «Фокус-режим» на 45 минут (предмет: математика)
2. Свет становится ярче (80%) и холоднее для концентрации
3. Уведомления на телефоне приглушаются через API оператора
4. **15:45** — Сессия завершена: +50 XP, перерыв 10 минут
5. Родители получают уведомление: «Миша专注ился 45 минут по математике!»

### Сценарий 3: «Вечерний ритуал»
1. **21:00** — Push-напоминание: «Пора начинать вечерний ритуал!»
2. **Шаг 1**: Проверка домашнего задания (фото в чат)
3. **Шаг 2**: Приглушение света в комнате до 30%
4. **Шаг 3**: Выключение развлекательных устройств (ТВ, консоль)
5. **Шаг 4**: Проверка дверей и окон (все закрыты ✅)
6. **Шаг 5**: Режим «Ночной страж» активирован
7. Интернет замедляется до минимума через оператора
8. **+30 XP** за завершение ритуала

### Сценарий 4: «Экстренная ситуация»
1. Датчик движения сработал в отсутствие семьи
2. Мгновенное SMS родителям через оператора
3. Камера делает снимок и сохраняет в облако
4. Сирена активируется автоматически
5. Миша получает уведомление: «Всё в порядке? Проверь дом»
6. **+20 XP** за быструю реакцию

### Сценарий 5: «Забытый утюг»
1. Умная розетка обнаруживает: утюг включен > 30 мин без движения
2. Push-уведомление: «⚠️ Утюг включен уже 30 минут!»
3. Миша может: выключить удалённо / напомнить маме / игнорировать
4. Если выключил удалённо: **+15 XP** «Эко-воин»
5. Если проигнорировал: SMS маме через 10 минут
