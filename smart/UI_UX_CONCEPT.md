# UI/UX Концепция «Ростелеком»

## 1. Дизайн-система

### Цветовая палитра
- **Основной фон**: `#1a1a2e` (глубокий тёмно-синий)
- **Вторичный фон**: `#16213e` (тёмно-синий)
- **Карточки**: `#242836` (тёмно-серый)
- **Акцент**: `#4fc3f7` (голубой)
- **Успех**: `#66bb6a` (зелёный)
- **Предупреждение**: `#ffa726` (оранжевый)
- **Ошибка**: `#ef5350` (красный)
- **Текст**: `#e8eaed` (белый), `#9aa0a6` (серый)

### Типографика
- **Заголовки**: 28px Bold
- **Подзаголовки**: 18px SemiBold
- **Основной текст**: 14px Regular
- **Мелкий текст**: 12px

### Иконки
- Lucide React — единый стиль
- Размеры: 16px, 20px, 24px

## 4. Интерактивные элементы

### Переключатели устройств
- Крупные, с анимацией
- Мгновенная обратная связь
- Цветовая индикация состояния

### Слайдеры яркости/температуры
- Градиентная полоса
- Числовое значение + визуальное изменение
- Шаг 1% для яркости, 0.5°C для температуры

### Карточки задач
- Цветовая кодировка по категориям
- Свайп для выполнения/отмены
- Анимация начисления XP

### Уведомления
- Правый верхний угол
- 3-5 секунд для обычных, до закрытия для важных
- Цветовая кодировка по типу

## 5. Анимации

### Переходы между страницами
- Горизонтальный слайд для основных разделов
- Плавное появление для модальных окон

### Микроанимации
- Переключатели: плавное изменение цвета
- Кнопки: лёгкое нажатие при клике
- Прогресс: плавное заполнение полос

### Геймификация
- Уровень: конфетти при повышении
- Очки: анимация начисления XP
- Достижения: всплывающая карточка

## 6. Адаптивность

| Экран | Поведение |
|-------|-----------|
| Десктоп (1200px+) | Полная боковая навигация, много колонок |
| Планшет (768-1199px) | Сворачиваемая навигация, 2 колонки |
| Мобильный (до 767px) | Нижняя навигация, 1 колонка |

## 7. Доступность

- Контрастность мин. 4.5:1
- Полная поддержка клавиатуры
- ARIA-атрибуты для скринридеров
- Масштабируемый текст до 200%
